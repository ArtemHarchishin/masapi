isBoolean()method
public static function isBoolean(str:String):Boolean

Retrieves if the specified String is a valid Boolean.

A valid boolean can be the following values :

  • true
  • false
  • Any Integer value
  • yes
  • no
All other values are not valid values. Not that the values are not case sensitive.

Parameters
str:String — The String to check.

Returns
Boolean — true if the String is a valid Boolean.

isNumber()method 
public static function isNumber(str:String):Boolean

Retrieves if the specified String is a valid Number.

A valid Number can be one of the following :

  • Any signed or unsigned Integer value
  • Any value that begins with a dot (.35)
  • Any value that ends with a dot (10.)

Parameters
str:String — The String to check.
