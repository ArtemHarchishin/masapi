Packagech.capi.core
Classpublic class ApplicationLoader

Manager of the loading of IApplicationFile objects. The purpose of this class is that it doesn't references statically the massload framework, so it can be used in sub-animations without heavy loading of classes. If you want to have a direct access to the ApplicationMassLoader used by the ApplicationLoader class, use the ApplicationContext.applicationMassLoader property value.
