getLoadableFile()method 
public function getLoadableFile(file:IApplicationFile):ILoadableFile

Retrieves a ILoadableFile from a IApplicationFile. This method uses the LoadableFileFactory to create the ILoadableFile using the url value of the IApplicationFile. A mapping is automatically established between the IApplicationFile and the ILoadableFile so that you can retrieve both of them using the getApplicationFile() or the getLoadableFile() method.
