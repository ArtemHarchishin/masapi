setDragDropBehaviorEnabled()method 
public function setDragDropBehaviorEnabled(value:Boolean = true, lockCenter:Boolean = false):void

Enables or disables the default drag and drop behavior. This method simply add or remove the listeners to the MouseEvent.MOUSE_DOWN and MouseEvent.MOUSE_UP events. Note that the DisplayObject must be already placed on the displayed tree !

Parameters
value:Boolean (default = true) — If true enableds the default drag and drop behavior.
 
lockCenter:Boolean (default = false) — Defines the value of the lockCenter parameter. If value false, this parameters doesn't change anything.
